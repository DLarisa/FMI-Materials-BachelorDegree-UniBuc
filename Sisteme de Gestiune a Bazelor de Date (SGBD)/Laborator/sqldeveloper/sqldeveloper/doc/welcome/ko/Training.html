SQL 및 PL/SQL 개발에 대한 Oracle 교육 경로
Oracle University에서 학습하기

Oracle은 SQL 및 PL/SQL 개발에 대한 포괄적인 교육을 제공하고 있습니다. 두 가지 학습 경로를 통해 SQL 및 PL/SQL 초보자 또는 숙련자를 대상으로 효과적인 교과 과정 계획을 제공합니다. 두 학습 경로에 나오는 교과 과정에서는 SQL Developer를 데이터베이스 개발을 위한 그래픽 도구로 사용할 예정입니다.

SQL 학습 경로에서는 대부분의 경우에 사용되는 필수 SQL 문을 학습합니다. 학습자는 최적의 성능을 낼 수 있도록 SQL 문을 튜닝하고 데이터 웨어하우징을 위해 분석 SQL 문을 사용하는 데 필요한 고급 정보를 얻을 수 있습니다.

PL/SQL 학습 경로h에서는 대부분의 경우에 사용되는 필수 PL/SQL 문과 고급 프로그래밍 항목을 학습합니다. 보안성이 높은 응용 프로그램을 작성하는 데 필요한 지식도 얻을 수 있습니다.

 

 

다양한 강의실 교육 과정과 함께 Oracle은 Oracle University Online Library를 통해 웹 기반 교육도 제공하고 있습니다. 이 편리한 서비스에는 Oracle의 최첨단 기술 교육을 다루는 다수의 제목이 포함되어 있습니다.
