SQL 和 PL/SQL 開發適用的 Oracle 教育訓練進程
從 Oracle University 學習

Oracle 提供 SQL 和 PL/SQL 開發適用的完整教育訓練。針對 SQL 和 PL/SQL 的新手及老鳥，一共有兩種提供有效課程計劃的學習進程。兩種學習進程中的課程都將使用 SQL Developer 作為資料庫開發的圖形工具。

在 SQL 學習進程中，您會學到一些常用的基本 SQL 敘述句。您也會習得調整 SQL 敘述句以達最佳效能及使用分析 SQL 敘述句進行資料倉儲的進階知識。

在 PL/SQL 學習進程中，您不僅會學到常用的基本 PL/SQL 敘述句，還會學到進階的程式設計主題。此外，您也會學到如何撰寫安全的應用程式。

 

 

除了教室課程之外，Oracle 還透過 Oracle University Online Library 提供網路教育訓練。這項便利的服務包括一系列教導 Oracle 具領先技術的課題。
